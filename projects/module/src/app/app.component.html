An implementation of
Prime Miner - Process Discovery using Prime Event Structures by R. Bergenthum published at ICPM 2019.
This implementation uses concurrency oracles to convert the input log into partial orders and then uses Petri net regions to synthesise a sequence of models from these partial orders according to the ideas presented in the paper.
